Última atualização: junho de 2025

1. Aceitação dos Termos

Ao acessar ou usar o Statify, você concorda com estes Termos de Uso. Se não concordar com algum dos termos, não utilize o serviço.

2. O que é o Statify

O Statify é um dashboard gratuito de estatísticas musicais que utiliza a API oficial do Spotify para exibir dados sobre seu histórico de escuta, artistas favoritos, músicas, álbuns e hábitos musicais.

O Statify não é afiliado, patrocinado ou endossado pelo Spotify. Spotify é uma marca registrada da Spotify AB.

3. Uso da API do Spotify

O Statify acessa os seguintes dados da sua conta Spotify, mediante sua autorização explícita:

  • Informações do perfil (nome, país, tipo de conta)
  • Top músicas e artistas
  • Histórico de reprodução recente
  • Músicas salvas / curtidas
  • Playlists públicas e privadas
  • Música tocando no momento

Todos os dados são obtidos diretamente do Spotify e processados localmente no seu navegador. Nenhum dado é armazenado em servidores externos.

4. Privacidade e Dados

O Statify não coleta, armazena ou compartilha seus dados pessoais em servidores próprios. Todo o processamento ocorre no navegador do usuário (client-side).

O token de acesso do Spotify é armazenado apenas no seu navegador (localStorage) e é usado exclusivamente para fazer requisições à API do Spotify em seu nome. Ele nunca é enviado a terceiros.

Ao sair da sessão ou limpar os dados do navegador, todas as informações são removidas automaticamente.

5. Código de Comparação entre Amigos

O recurso "Comparar com Amigo" gera um código compacto contendo IDs de músicas e artistas do Spotify do seu perfil. Este código pode ser compartilhado voluntariamente com outras pessoas.

Ao compartilhar seu código, você concorda que o destinatário poderá ver informações sobre suas músicas e artistas favoritos.

6. Limitações do Serviço

O Statify depende da API do Spotify para funcionar. Em caso de indisponibilidade da API, mudanças nas políticas do Spotify ou revogação do acesso, o serviço pode ser interrompido sem aviso prévio.

O serviço é oferecido "como está", sem garantias de disponibilidade contínua.

7. Uso Permitido

Você concorda em usar o Statify apenas para fins pessoais e não comerciais. É proibido:

  • Tentar burlar os sistemas de autenticação
  • Usar o serviço de forma automatizada (bots)
  • Reproduzir ou distribuir o código sem permissão
  • Usar o serviço para qualquer finalidade ilegal
